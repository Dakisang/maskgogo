# 😷 마스크고고(maskgogo) : 공적 마스크 검색

## ❓ 개요
- 정부제공 API를 이용한 공적마스크 재고확인 웹 애플리케이션 입니다. 
- 주변 위치의 공적마스크 재고를 보여줍니다.

## ⚙ 사용기술
- HTML, CSS, JavaScript(Jquery)
